How to use Shoe API:

Getting All Shoes

This gets every shoe in the database

Link: http://localhost:3000/api/v1/shoes

Example Result:

        Example Code: axios.get('http://localhost:3000/api/v1/shoes')

Getting One Shoe

This gets a single shoe from the database based on the id given

Link: http://localhost:3000/api/v1/shoes/:id

Example Result:

        Example Code: axios.get('http://localhost:3000/api/v1/shoes/61ba64a93c0b0bed8453e1b8')

Creating Shoes

This creates a shoe based with the information in the query

Link: http://localhost:3000/api/v1/shoes?name=[ProductName]&link=[ProductLink]&price=[ProductPrice]&featured=[ProductFeatured]&rating=[ProductRating]&createdAt=[ProductCreatedAt]&anime=[ProductAnime]

        Example Code: axios.post('http://localhost:3000/api/v1/shoes?name=Shoe&price=10&featured=false&rating=3&createdAt=[date]&anime=shoeAnime')

Deleting Shoes

This deletes a shoe based on the id

Link: http://localhost:3000/api/v1/shoes/:id

        Example Code: axios.delete('http://localhost:3000/api/v1/shoes/61ba64a93c0b0bed8453e1ba')

Updating Shoes

This updates individual shoes and it takes in an ID to know which one to update and a shoe object to update the selected one to

        Example Code: axios.patch('http://localhost:3000/api/v1/shoes/61ba64a93c0b0bed8453e1ba?name=test2&link=www.shoe.com&price=12&featured=true&rating=3.5&createdAt=2021-12-14T14:12:43.511Z&anime=Saiki%20K')
